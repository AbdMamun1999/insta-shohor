How Does Javascript work:

Javascript is an interpreted language.Normally javascript runs in any evironment like browser or any application program node JS.Javascript runs in where have javascript engine that exicutes javascript code line by line.The browsers use different engines to run JavaScript code.Node JS or google chrome uses V8 engine,Fire fox uses SpiderMonkey,Safari use Nitro engine.In engine,the first step is to parse the javascript code by a parser.It reads the javascript code line by line and check that syntax is correct.if the parser find any syntax error,it will throw an error and execution will stop.If the code is correct it create a data structure named Abstract sytax Tree.And this structue convert the code machine code.Then our machin can execute the code.Because when the code convert into machine code the machine understand the code.and finally we get the result

LocalStorage and SessionStorage

Web storage normally store data.Data must be stored in key-value pairs in web browser storage.there are to kind of web storage in web sotrage lilke Session storage and local storage.The data do not clear in Local atorage.It remains after close all tab and browser.Local storage does not work for any particular tab.It works for all tabs.Data in the local storage is manually deleted.On the other hand,Session storage work for any parcticular tab.If the tab or browser close,all data clear from session storage.Opening a new tab or window create a new session.when a tab duplicate,the new tab copies sessionStorage of the old tab.Data stored in sessionStorage is specific to the protocol of the page.
